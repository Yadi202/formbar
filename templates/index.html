LED Formbar (for classroom Formative Assessment and Management)

Teachers
  • To see the Virtual bar, go to /virtualbar
  • To change settings, go to /settings
  • To view a list of people who have requested help, go to /needshelp
  • To view and play our list of background music, go to /bgm, You can also stop this by going to /bgmstop
  • To Say something through the speakers, go to /say, with the Paramters phrase = {Phrase you want to say}

Students
  • To see the Virtual bar, go to Virtual Bar
  • To borrow a "Thumbs Panel" to interact with the bar, go to /tutd
  • To let the teacher know you need help, go to /help
  • To answer a quiz question, go to /quiz
  • To set the color of the LED bar, go to /color set the color either using hexdecial or RGB parameters
  • To pick a sound that can play throught the speakers, go to /sfx,You can also stop this by going to /sfxstop
  • To empty blocks from the MC chest, go to /emptyblocks
  • To add blocks into the MC chest, go to /sendblocks

Debugging
  • To get amount of pixels, go to /getpix

  • To set the health of boss, go to /perc, with the parametes amount = {percantageAmount}
